# Natural Language Processing (NLP) Project

## Project Overview

This project explores fundamental concepts of **Natural Language Processing (NLP)** using Python.  
The main objective is to demonstrate how basic programming structures and text-processing techniques can be applied to analyze and manipulate textual data.

---

## Technologies and Concepts Applied

- **Dictionary** – structured storage and word mapping  
- **Lists** – handling collections of text data  
- **Sets** – removing duplicates and working with unique values  
- **List Comprehension** – efficient and concise data processing  
- **Regular Expressions (Regex)** – pattern matching and text search  
- **Tokenization** – splitting text into meaningful units  

---

## Objectives

- Understand text preprocessing techniques  
- Apply NLP concepts without heavy frameworks  
- Practice Python data structures in real-world scenarios  
- Extract useful information from raw text
